The aim of the course was to master the fundamental methods and algorithms of Machine Learning, both practically and theoretically, focused on the mathematical foundations and practical deployment of ML workflows (supervised and unsupervised learning).
 - Predictive Modeling: Advanced regression and classification techniques with a focus on bias-variance tradeoff;
 - Pattern Recognition: Unsupervised learning methods for data clustering and structure discovery;
 - Model Optimization: Gained hands-on experience in feature engineering, cross-validation, and hyperparameter tuning.
